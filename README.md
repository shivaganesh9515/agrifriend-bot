# 🌱 AgriFriend Bot

WhatsApp AI farming consultant for agricultural communities.

## Features

- Answers farming, gardening, and plant questions
- Analyzes plant photos for diseases and health
- Remembers user conversations
- Proactively praises healthy plants and harvests
- Only responds to farming topics

## Quick Start

See **[SETUP.md](SETUP.md)** for step-by-step instructions.

## Requirements

- Hostinger VPS (Ubuntu)
- Node.js 22+
- Gemini API Key (free) — [Get here](https://aistudio.google.com/apikey)
- A spare WhatsApp number

## Tech Stack

- Node.js + TypeScript
- WhatsApp (Baileys)
- Google Gemini 2.0 Flash (AI)
- SQLite (user data)
- PM2 (process manager)
